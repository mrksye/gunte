# Contributing to gunte

Thanks for trying gunte! It's early (0.0.x) and lightly tested, so both bug reports and fixes are genuinely welcome.

## Two ways to help — both fine

**1. Just report it.** [Open an issue](https://github.com/mrksye/gunte/issues) and the maintainer will take it from there. A clear report is plenty.

**2. Fix it yourself.** Feel free to open an issue and send a PR for it too. No need to wait for permission — the maintainer will look at it and pull it in. Most PRs just get merged.

Either way, opening an issue first is nice so the context lives in one place, but it's not a hard gate.

## What to include in a report

- **Framework & adapter** — `gunte/core`, `gunte/native`, `gunte/solid`, `gunte/react`, `gunte/react-native`, or `gunte/vue`.
- **Platform** — browser + OS, or the mobile platform for React Native.
- **What happened vs. what you expected.**
- **A minimal reproduction** if you can (a small snippet or sandbox).
